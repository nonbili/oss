Nora 作為 Linux、macOS 與 Windows 的桌面應用程式發布。

## 桌面版新增功能

桌面版本增加了為大螢幕設計的佈局，包括可以更舒適地瀏覽多個時間軸的 Deck 檢視模式。

這使得桌面版更適合以下情況：

- 同時監控多個動態消息
- 為不同帳號或服務保持獨立的工作空間
- 希望 Nora 作為持續使用的日常瀏覽器，而非快速使用的行動工具

## 預期效果

桌面版保持與行動版相同的整體產品模式：

- 在一個應用程式中開啟支援的社群網站
- 在大螢幕上使用帳號隔離與瀏覽工具
- 安裝適用於 Linux、macOS 或 Windows 的原生軟體套件
