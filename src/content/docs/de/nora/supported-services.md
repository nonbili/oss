## Dienstspezifisches Verhalten

Einige Dienste verfügen über eine zusätzliche Nora-spezifische Handhabung für Login-Flows, Wiedergabe, Downloads oder Layout-Anpassungen. Derzeit ist dies dokumentiert für:

- Instagram
- X

Das bedeutet nicht, dass andere Dienste nicht unterstützt werden. Es bedeutet, dass viele Dienste über die gemeinsamen Browser-Funktionen von Nora funktionieren, während eine kleinere Gruppe zusätzliche standortspezifische Behandlung erhält.
