NouTube поддерживает пользовательский CSS для встроенных страниц YouTube и YouTube Music.

## Для чего хорош пользовательский CSS

- Скрытие элементов, которые вас не интересуют
- Повышение плотности страницы
- Настройка отступов или типографики
- Обход решений интерфейса сайта, которые вам мешают

## Рекомендуемый подход

Делайте изменения точечными и легко удаляемыми.

- Начните с одного или двух целевых правил
- По возможности отдавайте предпочтение стабильным селекторам
- Повторно тестируйте после изменений макета на стороне сайта

Поскольку NouTube оборачивает живые веб-ресурсы, пользовательский CSS может сломаться, когда YouTube изменит свою разметку.

## Перед настройкой

Убедитесь, что стандартное поведение приложения уже работает для:

- Воспроизведения
- Навигации
- Лент
- Загрузок

Когда база станет стабильной, постепенно добавляйте изменения CSS.
