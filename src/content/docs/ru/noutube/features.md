## Воспроизведение

- Воспроизведение без рекламы
- Фоновое воспроизведение
- Поддержка воспроизведения оригинального аудио
- Поддержка живого чата

## Библиотека и история

NouTube включает функции локального управления, которые не зависят от входа в аккаунт:

- Управление списком просмотра и музыкальной библиотекой без входа
- Управление историей просмотров без входа

## Открытие и ленты

- Читка RSS-лент для каналов YouTube
- Скрытие Shorts

## Загрузки

- Поддержка загрузки видео

## Настройка

- Поддержка пользовательского CSS

Если вы хотите настроить внешний вид и поведение встроенных сайтов, перейдите в раздел [Настройка](/ru/noutube/customization/).
