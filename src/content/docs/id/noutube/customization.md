NouTube mendukung CSS kustom untuk halaman YouTube dan YouTube Music yang disematkan.

## Kegunaan CSS Kustom

- Menyembunyikan elemen yang tidak Anda pedulikan
- Memperketat kepadatan halaman
- Menyesuaikan jarak atau tipografi
- Mengakali pilihan UI situs yang menghalangi Anda

## Pendekatan yang Disarankan

Jaga agar perubahan tetap sempit dan mudah dihapus.

- Mulai dengan satu atau dua aturan terfokus
- Lebih suka pemilih (selector) yang stabil jika memungkinkan
- Uji ulang setelah perubahan tata letak situs hulu

Karena NouTube membungkus properti web langsung, CSS kustom dapat rusak ketika YouTube mengubah markup-nya.

## Sebelum Anda Melakukan Kustomisasi

Pastikan perilaku aplikasi default sudah berfungsi untuk:

- Pemutaran
- Navigasi
- Feed
- Unduhan

Setelah garis dasar stabil, tambahkan perubahan CSS secara bertahap.
