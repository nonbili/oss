O NouTube tem uma versão desktop para Linux, macOS e Windows.

## O Que Esperar

A versão desktop mantém a mesma ideia geral do app mobile:

- YouTube e YouTube Music em um único app
- Instaladores nativos para Linux, macOS e Windows
- Uma janela desktop para sessões mais longas de escuta ou navegação

## Por que Usar o Desktop

Escolha a versão desktop se você deseja:

- Empacotamento desktop nativo
- Reprodução em janela
- Fluxos de instalação específicos da plataforma para Linux, macOS ou Windows

## Quando Preferir o Desktop

O desktop é a melhor opção quando você deseja o NouTube como um app de estação de trabalho diário, em vez de um player focado no celular.
