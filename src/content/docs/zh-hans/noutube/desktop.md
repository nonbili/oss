title: 桌面端
description: 在 Linux, macOS, 和 Windows 上使用 NouTube。
---
NouTube 为 Linux, macOS, 和 Windows 提供桌面端构建。

## 预期效果

桌面版本保持与移动端应用相同的整体理念：

- 在一个应用中同时使用 YouTube 和 YouTube Music
- 为 Linux, macOS, 和 Windows 提供原生安装程序
- 专为长时间聆听或浏览设计的桌面窗口

## 为什么选择桌面端

如果你希望获得以下体验，请选择桌面版本：

- 原生桌面包装
- 窗口化播放
- 针对 Linux, macOS, 或 Windows 的平台特定安装流程

## 何时优先使用桌面端

当你希望将 NouTube 作为日常工作站应用，而不是以手机为主的播放器时，桌面端是更好的选择。
