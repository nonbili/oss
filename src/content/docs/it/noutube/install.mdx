## Dopo l'Installazione

- Apri un link di YouTube o YouTube Music in NouTube se il tuo dispositivo ti chiede quale app utilizzare.
- Conferma che la riproduzione funzioni nel modo previsto prima di modificare le impostazioni avanzate.
- Continua con [Panoramica](/it/noutube/) per il contesto del prodotto, o [Funzionalità](/it/noutube/features/) per l'elenco completo delle capacità.
