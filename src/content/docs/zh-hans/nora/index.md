Nora 是面向社交网络服务的 Android、iOS 和桌面浏览器。

它保留各服务原本的网站，并加入账户隔离、拦截列表、下载、URL 清理和 CSS 自定义。

## 主要特性

- 无广告
- 多账户与独立 Cookie
- 媒体下载
- CSS 自定义

## 首次使用

1. 先打开你最常用的服务
2. 登录并确认浏览流程正常工作
3. 按需启用资料隔离和拦截列表
