## Comportamento Specifico per il Servizio

Alcuni servizi hanno una gestione extra specifica per Nora per i flussi di login, la riproduzione, i download o le regolazioni del layout. Al momento, questo è documentato per:

- Instagram
- X

Questo non significa che gli altri servizi non siano supportati. Significa che molti servizi funzionano attraverso le funzionalità browser condivise di Nora, mentre un set più piccolo riceve un trattamento aggiuntivo specifico per il sito.
