Nora est disponible en tant qu'application de bureau pour Linux, macOS et Windows.

## Ce que la version de bureau apporte

La version de bureau ajoute une mise en page conçue pour les écrans plus grands, incluant une vue "deck" pour parcourir plusieurs fils d'actualité plus confortablement.

Cela fait de la version de bureau un meilleur choix quand vous :

- Surveillez plusieurs flux à la fois
- Gardez des espaces de travail séparés ouverts pour différents comptes ou services
- Voulez que Nora fonctionne comme un navigateur quotidien persistant plutôt que comme un outil mobile rapide

## À quoi s'attendre

La version de bureau conserve le même modèle de produit global que la version mobile :

- Ouvrir les sites sociaux pris en charge dans une seule application
- Utiliser la séparation de compte et les outils de navigation sur un écran plus grand
- Installer des packages natifs pour Linux, macOS ou Windows
