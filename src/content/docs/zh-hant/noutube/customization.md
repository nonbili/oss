NouTube 支援為嵌入的 YouTube 與 YouTube Music 頁面使用自定義 CSS。

## 自定義 CSS 的用途

- 隱藏您不在意的元素
- 提高頁面內容密度
- 調整間距或字體
- 解決干擾您的網站 UI 設計問題

## 建議做法

保持變更範圍狹小且易於移除。

- 從一兩個針對性的規則開始
- 盡可能優先使用穩定的選取器（Selectors）
- 在上游網站佈局變更後重新測試

由於 NouTube 封裝了即時的網頁內容，當 YouTube 更改其標記（Markup）時，自定義 CSS 可能會失效。

## 在自定義之前

請確保應用程式的預設行為已能正常運作：

- 播放
- 導覽
- 訂閱內容
- 下載

一旦基礎功能穩定，再逐步添加 CSS 變更。
