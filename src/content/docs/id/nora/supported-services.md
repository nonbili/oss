- Facebook
- Instagram
- LinkedIn
- Reddit
- Threads
- TikTok
- Tumblr

## Perilaku Khusus Layanan

Beberapa layanan memiliki penanganan khusus Nora tambahan untuk alur masuk, pemutaran, unduhan, atau penyesuaian tata letak. Saat ini, hal tersebut didokumentasikan untuk:

- Instagram
- X

Itu tidak berarti layanan lain tidak didukung. Ini berarti banyak layanan berfungsi melalui fitur browser bersama Nora, sementara set yang lebih kecil mendapatkan perlakuan tambahan khusus situs.
