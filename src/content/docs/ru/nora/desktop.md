Nora выпускается как приложение для рабочего стола для Linux, macOS и Windows.

## Что добавляет десктопная версия

Десктопная версия добавляет макет, разработанный для больших экранов, включая режим колонок (deck view) для более комфортного просмотра нескольких лент одновременно.

Это делает десктопную версию более подходящей, когда вы:

- Отслеживаете несколько лент одновременно
- Держите открытыми отдельные рабочие пространства для разных аккаунтов или сервисов
- Хотите, чтобы Nora работала как постоянный ежедневный браузер, а не как быстрый мобильный инструмент

## Чего ожидать

Десктопная версия сохраняет ту же общую модель продукта, что и мобильная:

- Открывайте поддерживаемые социальные сайты в одном приложении
- Используйте разделение аккаунтов и инструменты просмотра на большом экране
- Устанавливайте нативные пакеты для Linux, macOS или Windows
