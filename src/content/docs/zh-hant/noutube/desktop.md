title: 桌面版
description: 在 Linux、macOS 與 Windows 上使用 NouTube。
---
NouTube 為 Linux、macOS 與 Windows 提供了桌面版本。

## 預期效果

桌面版本保持與行動版相同的整體理念：

- 在一個應用程式中整合 YouTube 與 YouTube Music
- 適用於 Linux、macOS 與 Windows 的原生安裝程式
- 適用於長時間聆聽或瀏覽的桌面視窗

## 為什麼使用桌面版

如果您需要以下功能，請選擇桌面版本：

- 原生桌面軟體套件
- 視窗化播放
- 適用於 Linux、macOS 或 Windows 的平台特定安裝流程

## 何時優先選擇桌面版

當您希望將 NouTube 作為日常工作站應用程式，而非以手機為主的播放器時，桌面版是更好的選擇。
