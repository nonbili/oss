## 再生

- 広告なしの再生
- バックグラウンド再生
- オリジナル音声再生のサポート
- ライブチャットのサポート

## ライブラリと履歴

NouTube には、ログインに依存しないローカル管理機能が含まれています：

- ログインなしでのウォッチリストとミュージックライブラリの管理
- ログインなしでの視聴履歴の管理

## 発見とフィード

- YouTube チャンネル用の RSS フィードリーダー
- Shorts の非表示

## ダウンロード

- 動画ダウンロードのサポート

## カスタマイズ

- カスタム CSS のサポート

埋め込みサイトの見た目や動作を調整したい場合は、[カスタマイズ](/ja/noutube/customization/) に進んでください。
