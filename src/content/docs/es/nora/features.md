## Aislamiento de cuentas

- Múltiples cuentas con almacenamiento de cookies independiente

Esta es la principal característica estructural que hace que Nora sea útil para las personas que necesitan una separación clara entre sus cuentas personales, de trabajo, de creador o secundarias.

## Bloqueo y limpieza

- Sin anuncios
- Soporte para listas de bloqueo EasyList y EasyPrivacy
- Limpieza de parámetros de consulta de URL de seguimiento

## Medios y lectura

- Descarga de imágenes
- Soporte de descarga de vídeo para Facebook, Instagram, TikTok y X
- Selección de texto
- Soporte de zoom

## Personalización

- Soporte de CSS personalizado

## Notas sobre las plataformas

La aplicación está disponible para móviles y escritorio, pero la idea del producto es la misma en cada plataforma: mantener el servicio web real eliminando las fricciones comunes en torno a los anuncios, el malabarismo de cuentas y el manejo de medios.
