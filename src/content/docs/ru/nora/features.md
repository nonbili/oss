## Изоляция аккаунтов

- Несколько аккаунтов с раздельным хранением куки

Это главная структурная функция, которая делает Nora полезной для людей, которым нужно четкое разделение между личными, рабочими, творческими или временными аккаунтами.

## Блокировка и очистка

- Без рекламы
- Поддержка списков блокировки EasyList и EasyPrivacy
- Очистка параметров отслеживания в URL-адресах

## Медиа и чтение

- Загрузка изображений
- Поддержка загрузки видео для Facebook, Instagram, TikTok и X
- Выделение текста
- Поддержка масштабирования

## Настройка

- Поддержка пользовательского CSS

## Примечания к платформам

Приложение выпускается для мобильных устройств и десктопа, но идея продукта одинакова на каждой платформе: сохранить настоящий веб-сервис, устранив при этом распространенные неудобства, связанные с рекламой, переключением аккаунтов и работой с медиафайлами.
