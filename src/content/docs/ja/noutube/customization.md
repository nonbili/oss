NouTube は、埋め込まれた YouTube および YouTube Music ページのカスタム CSS をサポートしています。

## カスタム CSS の利点

- 興味のない要素を非表示にする
- ページの密度を高める
- 余白やタイポグラフィを調整する
- 邪魔になるサイト UI を回避する

## 推奨されるアプローチ

変更は最小限に抑え、簡単に削除できるようにしてください。

- 1つまたは2つのターゲットを絞ったルールから始める
- 可能な限り安定したセレクターを使用する
- アップストリームのサイトレイアウト変更後に再テストする

NouTube はライブウェブサイトをラップしているため、YouTube がマークアップを変更するとカスタム CSS が機能しなくなる可能性があります。

## カスタマイズする前に

以下のデフォルトのアプリ動作が既に機能していることを確認してください：

- 再生
- ナビゲーション
- フィード
- ダウンロード

ベースラインが安定したら、CSS の変更を徐々に重ねてください。
