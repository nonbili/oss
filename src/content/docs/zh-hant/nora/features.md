## 帳號隔離

- 多個帳號具備獨立的 Cookie 存儲

這是 Nora 的核心結構化功能，對於需要在個人、工作、創作者或免洗帳號之間保持清晰隔離的使用者非常實用。

## 阻擋與清理

- 無廣告
- 支援 EasyList 與 EasyPrivacy 阻擋清單
- 清理追蹤 URL 查詢參數

## 媒體與閱讀

- 圖片下載
- 支援 Facebook、Instagram、TikTok 與 X 的影片下載
- 文字選取
- 縮放支援

## 自定義

- 支援自定義 CSS

## 平台說明

該應用程式在行動版與桌面版上都有提供，但其產品理念在每個平台上都是一致的：保留真實的網頁服務，同時消除常見的阻礙，例如廣告、帳號切換與媒體處理。
