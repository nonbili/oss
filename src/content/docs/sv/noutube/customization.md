NouTube stöder anpassad CSS för de inbäddade YouTube- och YouTube Music-sidorna.

## Vad Anpassad CSS Är Bra För

- Dölja element som du inte bryr dig om
- Öka sidans densitet
- Justera avstånd eller typografi
- Gå runt val i webbplatsens UI som är i vägen för dig

## Rekommenderat Tillvägagångssätt

Håll förändringarna små och lätta att ta bort.

- Börja med en eller två riktade regler
- Föredra stabila väljare där det är möjligt
- Testa igen efter layoutändringar på webbplatsen

Eftersom NouTube wrappar live-webbplatser kan anpassad CSS sluta fungera när YouTube ändrar sin kod.

## Innan Du Anpassar

Se till att appens standardbeteende redan fungerar för:

- Uppspelning
- Navigering
- Flöden
- Nedladdningar

När baslinjen är stabil kan du lägga till CSS-ändringar gradvis.
