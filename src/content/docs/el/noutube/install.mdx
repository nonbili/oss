## Μετά την Εγκατάσταση

- Ανοίξτε έναν σύνδεσμο YouTube ή YouTube Music στο NouTube εάν η συσκευή σας ρωτήσει ποια εφαρμογή να χρησιμοποιήσει.
- Επιβεβαιώστε ότι η αναπαραγωγή λειτουργεί όπως περιμένετε πριν αλλάξετε προχωρημένες ρυθμίσεις.
- Συνεχίστε με την [Επισκόπηση](/el/noutube/) για το πλαίσιο του προϊόντος, ή τα [Χαρακτηριστικά](/el/noutube/features/) για την πλήρη λίστα δυνατοτήτων.
