Αυτή η σελίδα συνοψίζει επαναλαμβανόμενες ερωτήσεις χρηστών από το δημόσιο GitHub issue tracker.

## Το κουμπί σύνδεσης στο desktop λείπει ή η σύνδεση φαίνεται να έχει κολλήσει

Εάν η εφαρμογή desktop δεν εμφανίζει ένα ορατό κουμπί `Sign in`, δοκιμάστε να ανοίξετε πρώτα το μενού `Your data in YouTube`. Στη συζήτηση του ζητήματος, αυτή η διαδρομή επέτρεψε στους επηρεαζόμενους χρήστες να φτάσουν στη ροή σύνδεσης, και ο συντηρητής αργότερα ανέφερε ότι το ορατό κουμπί θα έπρεπε να επιστρέψει στην έκδοση `v0.1.6`.

Πηγές:

- [Issue #46: desktop version never logs in always error](https://github.com/nonbili/NouTube/issues/46)

## Η σύνδεση στο desktop εξακολουθεί να αποτυγχάνει ακόμη και όταν ανοίγει η σελίδα σύνδεσης

Υπήρξαν αναφορές ότι ορισμένες ρυθμίσεις desktop εξακολουθούν να αποτυγχάνουν στη σύνδεση Google ακόμη και μετά τον καθαρισμό δεδομένων. Η συζήτηση του ζητήματος δεν κατέληξε σε μια επιβεβαιωμένη καθολική διόρθωση, επομένως η πρακτική καθοδήγηση είναι:

- ενημερώστε στην τελευταία έκδοση desktop
- καθαρίστε τα δεδομένα webview/προφίλ της εφαρμογής εάν έχετε ήδη προσπαθήσει να συνδεθείτε
- εάν εξακολουθεί να αποτυγχάνει, αναφέρετε το λειτουργικό σας σύστημα, τη μορφή πακέτου και στιγμιότυπα οθόνης στο GitHub

Πηγή:

- [Issue #103: Unable to login from desktop client](https://github.com/nonbili/NouTube/issues/103)

## Το YouTube Music δεν φορτώνει σωστά

Μια τεκμηριωμένη διαδρομή αντιμετώπισης προβλημάτων ήταν:

1. Βεβαιωθείτε ότι το `NouTube` είναι η προεπιλεγμένη εφαρμογή για το `music.youtube.com`.
2. Εάν είναι εγκατεστημένη η επίσημη εφαρμογή YouTube Music, αφαιρέστε πρώτα τον προεπιλεγμένο χειρισμό συνδέσμων της.
3. Εάν το πρόβλημα συνεχίζεται, δοκιμάστε ένα προσαρμοσμένο user agent από το `Settings > Tools > Custom user agent`.

Αυτή η καθοδήγηση προέρχεται απευθείας από τον συντηρητή στη συζήτηση του ζητήματος. Είναι μια παράκαμψη, όχι μια εγγυημένη διόρθωση για κάθε συσκευή.

Πηγή:

- [Issue #150: Noutube Music Error Message, Not Loading.](https://github.com/nonbili/NouTube/issues/150)

## Ορισμένες διαφημίσεις εξακολουθούν να εμφανίζονται

Υπήρξε μια αναφορά για σύντομες διαφημίσεις YouTube Premium που εμφανίζονται πριν από τα βίντεο. Ο συντηρητής παρέπεμψε τους χρήστες στην έκδοση `v0.4.9`, η οποία υποδηλώνει ότι οι διορθώσεις αποκλεισμού διαφημίσεων κυκλοφόρησαν σε μεταγενέστερες εκδόσεις. Εάν βλέπετε διαφημίσεις:

- ενημερώστε στην τελευταία έκδοση από το F-Droid ή το GitHub
- εάν το πρόβλημα επιμένει, ανοίξτε ένα νέο ζήτημα με την έκδοση της εφαρμογής και ένα στιγμιότυπο οθόνης

Πηγή:

- [Issue #117: Ad not blocked](https://github.com/nonbili/NouTube/issues/117)

## Το NouTube ξανανοίγει μια ουρά αναπαραγωγής αντί για την αρχική σελίδα

Εάν το NouTube συνεχίζει να επαναφέρει την αναπαραγωγή ή ανοίγει ξανά σε ένα παλιό στοιχείο αντί για την αρχική σελίδα, ελέγξτε αν το εικονίδιο της ουράς είναι ακόμα ενεργό. Στη συζήτηση του ζητήματος, ο συντηρητής εντόπισε μια ουρά που δεν είχε καθαριστεί ως τον λόγο που το NouTube συνέχιζε να προσπαθεί να συνεχίσει την αναπαραγωγή.

Πηγή:

- [Issue #138: Restoring playback](https://github.com/nonbili/NouTube/issues/138)
