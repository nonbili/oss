NouTube prend en charge le CSS personnalisé pour les pages YouTube et YouTube Music intégrées.

## À quoi sert le CSS personnalisé

- Masquer les éléments qui ne vous intéressent pas
- Augmenter la densité de la page
- Ajuster l'espacement ou la typographie
- Contourner les choix d'interface du site qui vous gênent

## Approche recommandée

Gardez les changements limités et faciles à supprimer.

- Commencez par une ou deux règles ciblées
- Préférez des sélecteurs stables autant que possible
- Testez à nouveau après les changements de mise en page du site source

Comme NouTube enveloppe des propriétés web en direct, le CSS personnalisé peut ne plus fonctionner lorsque YouTube modifie son balisage.

## Avant de personnaliser

Assurez-vous que le comportement par défaut de l'application fonctionne déjà pour :

- La lecture
- La navigation
- Les flux
- Les téléchargements

Une fois que la base est stable, ajoutez progressivement les modifications CSS.
