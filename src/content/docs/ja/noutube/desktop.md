NouTube には、Linux、macOS、および Windows 用のデスクトップビルドがあります。

## 期待できること

デスクトップ版は、モバイルアプリと同じコンセプトを維持しています：

- YouTube と YouTube Music を1つのアプリで
- Linux、macOS、および Windows 用のネイティブインストーラー
- 長時間のリスニングやブラウジングセッションに適したデスクトップウィンドウ

## なぜデスクトップ版を使うのか

以下を求める場合は、デスクトップ版を選択してください：

- ネイティブデスクトップパッケージ
- ウィンドウ表示での再生
- Linux、macOS、または Windows 用のプラットフォーム固有のインストールフロー

## デスクトップ版が適している場合

デスクトップ版は、電話メインのプレーヤーとしてではなく、日常的なワークステーションアプリとして NouTube を使用したい場合に適しています。
