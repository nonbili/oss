Nora mendukung CSS kustom sehingga Anda dapat membentuk kembali situs web layanan yang disematkannya.

## Penggunaan Umum

- Sembunyikan UI yang mengganggu
- Tingkatkan kepadatan untuk tata letak timeline yang padat
- Hapus chrome khusus layanan yang tidak pernah Anda gunakan
- Buat penyiapan multi-kolom desktop lebih mudah dipindai

## Jaga Perubahan Tetap Mudah Dipelihara

Karena Nora bergantung pada situs web langsung dari layanan eksternal, markup hulu dapat berubah tanpa peringatan.

Gunakan pendekatan konservatif:

- Lebih suka aturan kecil dan terfokus
- Uji ulang setelah desain ulang layanan
- Hindari override luas kecuali Anda mengendalikan dampaknya

## Pasangkan CSS dengan Profil

Jika Anda menggunakan profil terpisah untuk layanan atau peran yang berbeda, jaga agar penyesuaian CSS selaras dengan profil tersebut sehingga perilaku tetap dapat diprediksi.
