Nora unterstützt benutzerdefiniertes CSS, sodass Sie die Websites der eingebetteten Dienste umgestalten können.

## Häufige Anwendungen

- Ablenkende Benutzeroberfläche ausblenden
- Dichte für Timeline-lastige Layouts verbessern
- Dienstspezifische Elemente entfernen, die Sie nie verwenden
- Mehrspaltige Desktop-Setups einfacher scanbar machen

## Änderungen wartbar halten

Da Nora von den Live-Websites externer Dienste abhängt, kann sich das Markup jederzeit ohne Vorwarnung ändern.

Wählen Sie einen konservativen Ansatz:

- Bevorzugen Sie kleine, gezielte Regeln
- Testen Sie nach Neugestaltungen der Dienste erneut
- Vermeiden Sie umfassende Überschreibungen, es sei denn, Sie kontrollieren die Auswirkungen

## CSS mit Profilen kombinieren

Wenn Sie separate Profile für verschiedene Dienste oder Rollen verwenden, halten Sie die CSS-Anpassungen auf diese Profile abgestimmt, damit das Verhalten vorhersehbar bleibt.
