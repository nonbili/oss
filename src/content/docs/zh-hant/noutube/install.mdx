## 安裝後

- 如果您的裝置詢問應使用哪個應用程式，請在 NouTube 中開啟 YouTube 或 YouTube Music 連結。
- 在更改進階設定前，請確認播放效果符合您的預期。
- 請繼續閱讀 [概覽](/zh-hant/noutube/) 以獲取產品背景資訊，或閱讀 [功能](/zh-hant/noutube/features/) 以獲取完整功能清單。
