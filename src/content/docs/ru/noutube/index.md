NouTube — это приложение для Android и десктопа для YouTube и YouTube Music.

Оно оборачивает мобильные веб-приложения YouTube и YouTube Music и накладывает нативное поведение, включая блокировку рекламы, хуки управления воспроизведением, поддержку загрузки, RSS-ленты и управление библиотекой в офлайн-режиме.

## Основные возможности

- Без рекламы
- Фоновое воспроизведение
- Поддержка видео и музыки в одном приложении
- Управление библиотекой и историей без входа в аккаунт
- Загрузка видео
- Читка RSS-лент для каналов YouTube
- Поддержка живого чата
- Поддержка пользовательского CSS

## Платформы

- Android
- Десктоп

Десктопная сборка следует той же модели продукта, что и мобильное приложение, с нативными установщиками для больших экранов.

## Первый запуск

После установки практичная первая настройка выглядит так:

1. Откройте YouTube или YouTube Music внутри приложения.
2. Проверьте поведение воспроизведения, особенно фоновое воспроизведение, если это ваш основной сценарий использования.
3. Настройте параметры библиотеки, истории, лент и загрузок.
4. Добавляйте пользовательский CSS только после подтверждения того, что стандартный интерфейс вам подходит.

## Поддерживаемые ссылки

NouTube настроен на обработку этих хостов:

- `youtube.com`
- `www.youtube.com`
- `m.youtube.com`
- `music.youtube.com`
- `youtu.be`

Если ваше устройство спрашивает, какое приложение должно открыть поддерживаемую ссылку, выберите `NouTube`.

## Как это работает

NouTube оборачивает веб-приложения, а не перестраивает YouTube с нуля. На практике это означает:

- Вы сохраняете официальный поток и макет сайта
- Приложение добавляет блокировку рекламы и нативные хуки воспроизведения
- Фоновое воспроизведение и интеграция с устройством работают лучше, чем при использовании сайта в обычном браузере
