Nora — это браузер для Android, iOS и десктопа, оптимизированный для работы с социальными сетями.

Он сохраняет оригинальные веб-сайты сервисов, добавляя нативные удобства, такие как раздельное хранение аккаунтов, списки блокировки, инструменты для загрузки, очистка URL-адресов и настройка CSS.

## Основные возможности

- Без рекламы
- Несколько аккаунтов с раздельным хранением куки
- Поддержка списков блокировки EasyList и EasyPrivacy
- Загрузка медиафайлов
- Выделение текста и масштабирование
- Удаление параметров отслеживания из URL-адресов
- Поддержка пользовательского CSS

## Первый запуск

Разумная первая настройка:

1. Откройте сервис, которым вы пользуетесь чаще всего.
2. Войдите в систему и убедитесь, что веб-интерфейс работает нормально внутри Nora.
3. Создайте дополнительные профили, если вам нужно строгое разделение между аккаунтами.
4. Включите списки блокировки и проверьте загрузку медиа в сервисах, которые вас интересуют.
5. Добавляйте пользовательский CSS только после того, как основной процесс просмотра станет стабильным.

## Как это работает

Nora оборачивает веб-сайты сервисов вместо того, чтобы перестраивать каждую сеть как нативный клиент.

Такой подход сохраняет привычный вид сайтов, позволяя Nora добавлять:

- Блокировку рекламы
- Изоляцию куки между аккаунтами
- Процессы загрузки медиафайлов
- Улучшения для конкретных сервисов
