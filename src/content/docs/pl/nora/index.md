Nora to przeglądarka na Androida, iOS i desktop, zoptymalizowana pod kątem serwisów społecznościowych.

Zachowuje oryginalne strony internetowe serwisów, dodając jednocześnie natywne udogodnienia, takie jak oddzielne przechowywanie kont, listy blokowania, narzędzia do pobierania, czyszczenie adresów URL i dostosowywanie CSS.

## Główne Funkcje

- Brak reklam
- Wiele kont z oddzielnym przechowywaniem plików cookie
- Obsługa list blokowania EasyList i EasyPrivacy
- Pobieranie mediów
- Zaznaczanie tekstu i powiększanie
- Usuwanie parametrów śledzenia z adresów URL
- Obsługa niestandardowego CSS

## Pierwsze Uruchomienie

Rozsądna pierwsza konfiguracja to:

1. Otwórz serwis, z którego korzystasz najczęściej.
2. Zaloguj się i sprawdź, czy przeglądanie wewnątrz Nora działa normalnie.
3. Utwórz dodatkowe profile, jeśli chcesz ścisłej separacji między kontami.
4. Włącz listy blokowania i przetestuj pobieranie mediów w serwisach, na których Ci zależy.
5. Dodaj własny CSS dopiero wtedy, gdy podstawowe przeglądanie będzie stabilne.

## Jak To Działa

Nora stanowi powłokę dla stron internetowych serwisów, zamiast budować każdą sieć od nowa jako natywnego klienta.

Takie podejście sprawia, że strony pozostają znajome, a Nora może dodać:

- Blokowanie reklam
- Izolację ciasteczek między kontami
- Funkcje pobierania mediów
- Ulepszenia specyficzne dla danego serwisu
