Nora は Linux、macOS、および Windows 用のデスクトップアプリとして提供されています。

## デスクトップ版の追加機能

デスクトップ版では、複数のタイムラインをより快適に閲覧できるデッキビューなど、大画面向けに設計されたレイアウトが追加されています。

これにより、以下のような場合にデスクトップ版が最適です：

- 複数のフィードを同時に監視する
- 異なるアカウントやサービス用に個別のワークスペースを開いたままにする
- クイックなモバイルツールとしてではなく、常に使用するメインブラウザとして Nora を機能させたい

## 期待できること

デスクトップ版は、モバイル版と同じ製品モデルを維持しています：

- 1つのアプリ内でサポートされているソーシャルサイトを開く
- 大画面でアカウントの分離とブラウジングツールを使用する
- Linux、macOS、または Windows 用のネイティブパッケージをインストールする
