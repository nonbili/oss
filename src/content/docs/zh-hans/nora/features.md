## 账号隔离

- 具有独立 Cookie 存储的多账号支持

这是使 Nora 变得实用的主要结构性功能，适合需要清晰分离个人、工作、创作者或临时账号的用户。

## 拦截与清理

- 无广告
- 支持 EasyList 和 EasyPrivacy 拦截列表
- 跟踪 URL 查询参数清理

## 媒体与阅读

- 图片下载
- 支持 Facebook, Instagram, TikTok, 和 X 的视频下载
- 文本选择
- 缩放支持

## 个性化

- 支持自定义 CSS

## 平台说明

该应用发布于移动端和桌面端，但在每个平台上的产品理念是相同的：保留真实的 Web 服务，同时消除广告、账号切换和媒体处理等常见摩擦。
