Nora 支援自定義 CSS，因此您可以重新調整其嵌入的服務網站。

## 常見用途

- 隱藏干擾視線的 UI
- 提高時間軸密集型佈局的內容密度
- 移除您從不使用的特定服務介面
- 使桌面版多欄設定更易於瀏覽

## 保持變更的可維護性

由於 Nora 依賴於外部服務的即時網站，上游的標記（Markup）可能會在不經通知的情況下發生變化。

請採取保守的做法：

- 優先使用小型、具針對性的規則
- 在服務重新設計後重新測試
- 除非您能控制其影響，否則請避免廣泛的覆蓋

## 將 CSS 與設定檔配合使用

如果您為不同的服務或角色使用獨立的設定檔，請讓 CSS 調整與這些設定檔保持一致，以確保行為可預測。
