NouTube cuenta con una versión de escritorio para Linux, macOS y Windows.

## Qué esperar

La versión de escritorio mantiene la misma idea general que la aplicación móvil:

- YouTube y YouTube Music en una sola aplicación
- Instaladores nativos para Linux, macOS y Windows
- Una ventana de escritorio para sesiones de escucha o navegación más largas

## Por qué usar la versión de escritorio

Elige la versión de escritorio si quieres:

- Paquetes nativos para escritorio
- Reproducción en ventana
- Flujos de instalación específicos de la plataforma para Linux, macOS o Windows

## Cuándo preferir el escritorio

El escritorio es la mejor opción cuando quieres NouTube como una aplicación diaria de trabajo en lugar de un reproductor pensado primero para el teléfono.
