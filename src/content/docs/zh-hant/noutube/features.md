## 播放

- 無廣告播放
- 背景播放
- 支援原始音訊播放
- 即時聊天支援

## 媒體庫與歷史紀錄

NouTube 包含不依賴登入的本地管理功能：

- 無需登入即可管理觀看清單與音樂庫
- 無需登入即可管理觀看歷史紀錄

## 探索與訂閱

- 適用於 YouTube 頻道的 RSS 訂閱閱讀器
- 隱藏 Shorts 短片

## 下載

- 支援影片下載

## 自定義

- 自定義 CSS 支援

如果您想調整嵌入網站的外觀與行為，請繼續閱讀 [自定義](/zh-hant/noutube/customization/)。
