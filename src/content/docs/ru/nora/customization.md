title: Настройка
description: CSS и настройки просмотра в Nora.
---
Nora поддерживает пользовательский CSS, поэтому вы можете изменять вид веб-сайтов сервисов, которые она встраивает.

## Распространенные способы использования

- Скрыть отвлекающие элементы интерфейса
- Улучшить плотность макетов с насыщенными лентами
- Удалить специфические элементы сервиса, которыми вы никогда не пользуетесь
- Сделать многоколонные макеты на десктопе более удобными для сканирования

## Сохраняйте изменения поддерживаемыми

Поскольку Nora зависит от живых веб-сайтов внешних сервисов, разметка может измениться без предупреждения.

Используйте консервативный подход:

- Отдавайте предпочтение небольшим целевым правилам
- Повторно тестируйте после редизайна сервисов
- Избегайте широких переопределений, если вы не контролируете последствия

## Сочетайте CSS с профилями

Если вы используете отдельные профили для разных сервисов или ролей, следите за тем, чтобы настройки CSS соответствовали этим профилям, чтобы поведение оставалось предсказуемым.
