Το Nora είναι ένα πρόγραμμα περιήγησης για Android, iOS και desktop βελτιστοποιημένο για υπηρεσίες κοινωνικών δικτύων.

Διατηρεί τους αρχικούς ιστότοπους των υπηρεσιών προσθέτοντας native ευκολίες όπως ξεχωριστή αποθήκευση λογαριασμών, λίστες αποκλεισμού, εργαλεία λήψης, καθαρισμό URL και προσαρμογή CSS.

## Κύρια Σημεία

- Χωρίς διαφημίσεις
- Πολλαπλοί λογαριασμοί με ξεχωριστή αποθήκευση cookies
- Υποστήριξη λιστών αποκλεισμού EasyList και EasyPrivacy
- Λήψεις πολυμέσων
- Επιλογή κειμένου και ζουμ
- Αφαίρεση παραμέτρων παρακολούθησης από τα URL
- Υποστήριξη προσαρμοσμένου CSS

## Πρώτη Εκτέλεση

Μια λογική πρώτη ρύθμιση είναι:

1. Ανοίξτε την υπηρεσία που χρησιμοποιείτε συχνότερα.
2. Συνδεθείτε και επαληθεύστε ότι η ροή ιστού λειτουργεί κανονικά μέσα στο Nora.
3. Δημιουργήστε πρόσθετα προφίλ εάν θέλετε αυστηρό διαχωρισμό μεταξύ των λογαριασμών.
4. Ενεργοποιήστε τις λίστες αποκλεισμού και δοκιμάστε τη συμπεριφορά λήψης πολυμέσων στις υπηρεσίες που σας ενδιαφέρουν.
5. Προσθέστε προσαρμοσμένο CSS μόνο αφού η βασική ροή περιήγησης είναι σταθερή.

## Πώς Λειτουργεί

Το Nora περιβάλλει τους ιστότοπους των υπηρεσιών αντί να ξαναχτίζει κάθε δίκτυο ως native client.

Αυτή η προσέγγιση διατηρεί τους ιστότοπους οικείους, επιτρέποντας στο Nora να προσθέτει:

- Αποκλεισμό διαφημίσεων
- Απομόνωση cookies μεταξύ λογαριασμών
- Ροές λήψης πολυμέσων
- Βελτιώσεις ειδικά για κάθε υπηρεσία
