O NouTube suporta CSS personalizado para as páginas incorporadas do YouTube e YouTube Music.

## Para Que o CSS Personalizado é Bom

- Ocultar elementos que não lhe interessam
- Aumentar a densidade da página
- Ajustar espaçamento ou tipografia
- Contornar escolhas de UI do site que atrapalham você

## Abordagem Recomendada

Mantenha as alterações restritas e fáceis de remover.

- Comece com uma ou duas regras direcionadas
- Prefira seletores estáveis sempre que possível
- Teste novamente após alterações de layout do site de origem

Como o NouTube envolve propriedades web ao vivo, o CSS personalizado pode quebrar quando o YouTube altera sua marcação.

## Antes de Customizar

Certifique-se de que o comportamento padrão do app já funcione para:

- Reprodução
- Navegação
- Feeds
- Downloads

Assim que a base estiver estável, adicione as alterações de CSS gradualmente.
