NouTube unterstützt benutzerdefiniertes CSS für die eingebetteten YouTube- und YouTube Music-Seiten.

## Wofür benutzerdefiniertes CSS gut ist

- Ausblenden von Elementen, die Sie nicht interessieren
- Erhöhung der Seitendichte
- Anpassung von Abständen oder Typografie
- Umgehung von UI-Entscheidungen der Website, die Sie stören

## Empfohlener Ansatz

Halten Sie Änderungen eng begrenzt und leicht entfernbar.

- Beginnen Sie mit einer oder zwei gezielten Regeln
- Bevorzugen Sie stabile Selektoren, wo immer möglich
- Testen Sie nach Layout-Änderungen der ursprünglichen Website erneut

Da NouTube Live-Web-Inhalte einbindet, kann benutzerdefiniertes CSS nicht mehr funktionieren, wenn YouTube sein Markup ändert.

## Vor der Anpassung

Stellen Sie sicher, dass das Standardverhalten der App bereits funktioniert für:

- Wiedergabe
- Navigation
- Feeds
- Downloads

Sobald die Basis stabil ist, fügen Sie CSS-Änderungen schrittweise hinzu.
