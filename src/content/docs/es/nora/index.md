Nora es un navegador para Android, iOS y escritorio optimizado para servicios de redes sociales.

Mantiene los sitios web originales y añade separacion de cuentas, listas de bloqueo, descargas, limpieza de URL y personalizacion CSS.

## Funciones principales

- Sin anuncios
- Varias cuentas con cookies separadas
- Descargas de medios
- Personalizacion CSS

## Primeros pasos

1. Abre primero el servicio que mas usas
2. Inicia sesion y comprueba que la navegacion funcione con normalidad
3. Activa perfiles separados y blocklists si los necesitas
