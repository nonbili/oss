NouTube dispose d'une version de bureau pour Linux, macOS et Windows.

## À quoi s'attendre

La version de bureau conserve la même idée générale que l'application mobile :

- YouTube et YouTube Music dans une seule application
- Programmes d'installation natifs pour Linux, macOS et Windows
- Une fenêtre de bureau pour des sessions d'écoute ou de navigation plus longues

## Pourquoi utiliser la version de bureau

Choisissez la version de bureau si vous voulez :

- Un packaging natif pour bureau
- Une lecture en fenêtre
- Des flux d'installation spécifiques à la plateforme pour Linux, macOS ou Windows

## Quand préférer la version de bureau

La version de bureau est mieux adaptée lorsque vous voulez utiliser NouTube comme une application de travail quotidienne plutôt que comme un lecteur mobile.
