## Μετά την Εγκατάσταση

- Ανοίξτε έναν από τους συνηθισμένους κοινωνικούς σας συνδέσμους στο Nora εάν η συσκευή σας ρωτήσει ποια εφαρμογή να χρησιμοποιήσει.
- Συνδεθείτε στις υπηρεσίες που χρησιμοποιείτε περισσότερο και επιβεβαιώστε ότι η κανονική περιήγηση λειτουργεί πρώτα.
- Συνεχίστε με την [Επισκόπηση](/el/nora/) για το πλαίσιο του προϊόντος, ή τα [Χαρακτηριστικά](/el/nora/features/) για την πλήρη λίστα δυνατοτήτων.
