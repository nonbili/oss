## 安裝後

- 如果您的裝置詢問應使用哪個應用程式，請在 Nora 中開啟您常用的社群連結。
- 先登入您最常用的服務，並確認一般瀏覽功能正常。
- 請繼續閱讀 [概覽](/zh-hant/nora/) 以獲取產品背景資訊，或閱讀 [功能](/zh-hant/nora/features/) 以獲取完整功能清單。
