Nora finns som en desktop-app för Linux, macOS och Windows.

## Vad Desktop Lägger Till

Desktop-versionen lägger till en layout designad för större skärmar, inklusive en däckvy (deck view) för att surfa i flera tidslinjer mer bekvämt.

Det gör desktop till ett bättre val när du:

- Övervakar flera flöden samtidigt
- Håller separata arbetsytor öppna för olika konton eller tjänster
- Vill att Nora ska fungera som en permanent daglig webbläsare snarare än ett snabbt mobilverktyg

## Vad Du Kan Förvänta Dig

Desktop behåller samma övergripande produktmodell som mobil:

- Öppna sociala webbplatser som stöds i en app
- Använd kontoseparation och webbläsarverktyg på en större skärm
- Installera native paket för Linux, macOS eller Windows
