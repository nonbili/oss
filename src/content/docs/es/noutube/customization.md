title: Personalización
description: Notas sobre CSS personalizado y comportamiento para NouTube.
---
NouTube admite CSS personalizado para las páginas integradas de YouTube y YouTube Music.

## Para qué sirve el CSS personalizado

- Ocultar elementos que no te interesan
- Aumentar la densidad de la página
- Ajustar el espaciado o la tipografía
- Solucionar decisiones de la interfaz de usuario del sitio que te estorben

## Enfoque recomendado

Haz que los cambios sean limitados y fáciles de eliminar.

- Empieza con una o dos reglas específicas
- Prefiere selectores estables siempre que sea posible
- Vuelve a realizar pruebas después de los cambios en el diseño del sitio original

Dado que NouTube envuelve propiedades web en vivo, el CSS personalizado puede dejar de funcionar cuando YouTube cambie su marcado.

## Antes de personalizar

Asegúrate de que el comportamiento predeterminado de la aplicación ya funciona para:

- Reproducción
- Navegación
- Feeds
- Descargas

Una vez que la base sea estable, añade los cambios de CSS gradualmente.
