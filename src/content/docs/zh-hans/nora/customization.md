title: 自定义
description: Nora 中的 CSS 和浏览调整。
---
Nora 支持自定义 CSS，因此你可以重塑它所嵌入的服务网站。

## 常见用途

- 隐藏干扰 UI
- 提高时间线密集型布局的密度
- 移除你从不使用的特定服务界面
- 使桌面端多列设置更易于浏览

## 保持更改的可维护性

由于 Nora 依赖于外部服务的实时网站，上游标记可能会在没有警告的情况下发生变化。

请采取保守的方法：

- 优先使用细微、有针对性的规则
- 在服务重新设计后重新测试
- 除非你能控制后果，否则避免广泛的覆盖

## CSS 与配置文件配对

如果你为不同的服务或角色使用独立的配置文件，请保持 CSS 调整与这些配置文件对齐，以便行为保持可预测。
