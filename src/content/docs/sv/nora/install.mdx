## Efter Installation

- Öppna en av dina vanliga sociala länkar i Nora om din enhet frågar vilken app som ska användas.
- Logga in på de tjänster du använder mest och bekräfta att normal surfning fungerar först.
- Fortsätt med [Översikt](/sv/nora/) för produktkontext, eller [Funktioner](/sv/nora/features/) för den fullständiga listan över funktioner.
