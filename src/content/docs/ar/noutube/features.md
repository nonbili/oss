## التشغيل

- تشغيل بدون إعلانات
- تشغيل في الخلفية
- دعم تشغيل الصوت الأصلي
- دعم الدردشة المباشرة

## المكتبة والسجل

يتضمن NouTube ميزات إدارة محلية لا تعتمد على تسجيل الدخول:

- إدارة قائمة المشاهدة ومكتبة الموسيقى بدون تسجيل دخول
- إدارة سجل المشاهدة بدون تسجيل دخول

## الاكتشاف والخلاصات

- قارئ خلاصة RSS لقنوات YouTube
- إخفاء الفيديوهات القصيرة (Shorts)

## التنزيلات

- دعم تنزيل الفيديو

## التخصيص

- دعم CSS مخصص
