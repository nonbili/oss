## Isolasi Akun

- Beberapa akun dengan penyimpanan cookie terpisah

Ini adalah fitur struktural utama yang membuat Nora berguna bagi orang yang membutuhkan pemisahan bersih antara akun pribadi, pekerjaan, kreator, atau anonim.

## Pemblokiran dan Pembersihan

- Tanpa iklan
- Dukungan blocklist EasyList dan EasyPrivacy
- Pembersihan parameter kueri URL pelacakan

## Media dan Membaca

- Unduhan gambar
- Dukungan unduhan video untuk Facebook, Instagram, TikTok, dan X
- Pemilihan teks
- Dukungan zoom

## Kustomisasi

- Dukungan CSS kustom

## Catatan Platform

Aplikasi ini hadir di ponsel dan desktop, tetapi ide produknya sama di setiap platform: mempertahankan layanan web asli sambil menghilangkan hambatan umum seputar iklan, pengelolaan akun, dan penanganan media.
