## アカウントの分離

- 個別の Cookie ストレージによるマルチアカウント対応

これは、個人用、仕事用、クリエイター用、または使い捨て用のアカウントを明確に分離する必要がある人々にとって、Nora を有用にする主な構造的機能です。

## ブロックとクリーンアップ

- 広告なし
- EasyList および EasyPrivacy ブロックリストのサポート
- トラッキング URL クエリパラメータのクリーンアップ

## メディアと閲覧

- 画像のダウンロード
- Facebook、Instagram、TikTok、および X の動画ダウンロードのサポート
- テキスト選択
- ズームのサポート

## カスタマイズ

- カスタム CSS のサポート

## プラットフォームに関するメモ

アプリはモバイルとデスクトップで提供されていますが、製品のコンセプトはすべてのプラットフォームで同じです。広告、アカウントの切り替え、メディアの取り扱いに関する一般的な不便さを取り除きつつ、本物のウェブサービスを維持することです。
