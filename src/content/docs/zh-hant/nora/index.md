Nora 是一款為社群網路服務優化的 Android、iOS 與桌面瀏覽器。

它保留了原始的服務網站，同時增加了原生便利功能，如獨立的帳號存儲、黑名單、下載工具、URL 清理與 CSS 自定義。

## 亮點

- 無廣告
- 多個帳號具備獨立的 Cookie 存儲
- 支援 EasyList 與 EasyPrivacy 阻擋清單
- 媒體下載
- 文字選取與縮放
- 移除 URL 中的追蹤查詢參數
- 自定義 CSS 支援

## 初次運行

建議的初次設定如下：

1. 開啟您最常用的服務。
2. 登入並在 Nora 內驗證網頁流程是否正常運作。
3. 如果您希望在帳號之間保持嚴格隔離，請建立額外的設定檔（Profiles）。
4. 在您關心的服務上啟用黑名單並測試媒體下載行為。
5. 僅在基準瀏覽流程穩定後，再添加自定義 CSS。

## 運作原理

Nora 是封裝服務網站，而非將每個網路重建為原生客戶端。

這種方法保持了網站的熟悉感，同時讓 Nora 能夠增加：

- 廣告阻擋
- 帳號之間的 Cookie 隔離
- 媒體下載流程
- 針對特定服務的增強功能
