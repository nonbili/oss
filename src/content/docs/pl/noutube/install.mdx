## Po Instalacji

- Otwórz link do YouTube lub YouTube Music w NouTube, jeśli urządzenie zapyta, której aplikacji użyć.
- Zanim zmienisz zaawansowane ustawienia, potwierdź, że odtwarzanie działa zgodnie z Twoimi oczekiwaniami.
- Przejdź do [Przegląd](/pl/noutube/), aby poznać kontekst produktu, lub [Funkcje](/pl/noutube/features/), aby zobaczyć pełną listę możliwości.
