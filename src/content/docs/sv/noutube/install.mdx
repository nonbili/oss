## Efter Installation

- Öppna en YouTube- eller YouTube Music-länk i NouTube om din enhet frågar vilken app som ska användas.
- Bekräfta att uppspelningen fungerar som du förväntar dig innan du ändrar avancerade inställningar.
- Fortsätt med [Översikt](/sv/noutube/) för produktkontext, eller [Funktioner](/sv/noutube/features/) för den fullständiga listan över funktioner.
