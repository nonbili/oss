NouTube 支持对嵌入的 YouTube 和 YouTube Music 页面使用自定义 CSS。

## 自定义 CSS 的作用

- 隐藏你不在意的元素
- 收紧页面密度
- 调整间距或排版
- 绕过阻碍你使用的站点 UI 设计

## 推荐方法

保持更改范围狭窄且易于移除。

- 从一两条有针对性的规则开始
- 尽可能优先使用稳定的选择器
- 在上游站点布局更改后重新测试

由于 NouTube 包装的是实时 Web 属性，当 YouTube 更改其标记时，自定义 CSS 可能会失效。

## 在自定义之前

确保应用的默认行为在以下方面已经正常工作：

- 播放
- 导航
- Feed
- 下载

一旦基准线稳定，再逐步增加 CSS 更改。
