Nora adalah browser Android, iOS, dan desktop yang dioptimalkan untuk layanan jejaring sosial.

Aplikasi ini mempertahankan situs web layanan asli sambil menambahkan kenyamanan native seperti penyimpanan akun terpisah, blocklist, alat pengunduhan, pembersihan URL, dan kustomisasi CSS.

## Sorotan

- Tanpa iklan
- Beberapa akun dengan penyimpanan cookie terpisah
- Dukungan blocklist EasyList dan EasyPrivacy
- Unduhan media
- Pilih teks dan zoom
- Hapus parameter kueri pelacakan dari URL
- Dukungan CSS kustom

## Penggunaan Pertama

Penyiapan awal yang masuk akal adalah:

1. Buka layanan yang paling sering Anda gunakan.
2. Masuk dan verifikasi bahwa alur web berfungsi normal di dalam Nora.
3. Buat profil tambahan jika Anda ingin pemisahan ketat antar akun.
4. Aktifkan blocklist dan uji perilaku unduhan media pada layanan yang Anda pedulikan.
5. Tambahkan CSS kustom hanya setelah alur penjelajahan dasar stabil.

## Cara Kerjanya

Nora membungkus situs web layanan alih-alih membangun kembali setiap jaringan sebagai klien native.

Pendekatan tersebut menjaga situs tetap familiar sambil membiarkan Nora menambahkan:

- Pemblokiran iklan
- Isolasi cookie antar akun
- Alur pengunduhan media
- Peningkatan khusus layanan
