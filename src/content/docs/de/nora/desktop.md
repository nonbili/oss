Nora wird als Desktop-App für Linux, macOS und Windows ausgeliefert.

## Was Desktop hinzufügt

Die Desktop-Version bietet ein Layout, das für größere Bildschirme konzipiert ist, einschließlich einer Deck-Ansicht zum bequemeren Durchsuchen mehrerer Timelines.

Das macht Desktop zur besseren Wahl, wenn Sie:

- Mehrere Feeds gleichzeitig überwachen
- Getrennte Arbeitsbereiche für verschiedene Konten oder Dienste offen halten
- Nora als beständigen täglichen Browser anstelle eines schnellen mobilen Tools nutzen möchten

## Was zu erwarten ist

Desktop behält das gleiche Produktmodell wie Mobile bei:

- Unterstützte soziale Netzwerke in einer App öffnen
- Kontentrennung und Browsing-Tools auf einem größeren Bildschirm nutzen
- Native Pakete für Linux, macOS oder Windows installieren
