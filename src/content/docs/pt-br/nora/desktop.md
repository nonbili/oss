O Nora está disponível como um aplicativo de desktop para Linux, macOS e Windows.

## O Que o Desktop Adiciona

A versão desktop adiciona um layout projetado para telas maiores, incluindo uma visualização em deck para navegar por múltiplas timelines com mais conforto.

Isso torna o desktop a melhor opção quando você:

- Monitora vários feeds ao mesmo tempo
- Mantém espaços de trabalho separados abertos para diferentes contas ou serviços
- Quer que o Nora funcione como um navegador diário persistente em vez de uma ferramenta móvel rápida

## O Que Esperar

O desktop mantém o mesmo modelo geral de produto do mobile:

- Abra sites sociais suportados em um único app
- Use separação de conta e ferramentas de navegação em uma tela maior
- Instale pacotes nativos para Linux, macOS ou Windows
