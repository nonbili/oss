## После установки

- Откройте одну из ваших обычных социальных ссылок в Nora, если устройство спросит, какое приложение использовать.
- Войдите в сервисы, которыми вы пользуетесь чаще всего, и сначала подтвердите нормальную работу просмотра.
- Продолжите с [Обзора](/ru/nora/) для ознакомления с продуктом или [Функций](/ru/nora/features/) для получения полного списка возможностей.
