Nora se distribuye como una aplicación de escritorio para Linux, macOS y Windows.

## Qué añade el escritorio

La versión de escritorio añade un diseño pensado para pantallas más grandes, incluyendo una vista de "deck" para navegar por múltiples líneas de tiempo de forma más cómoda.

Eso hace que el escritorio sea la mejor opción cuando:

- Monitorizas varios feeds a la vez
- Mantienes abiertos espacios de trabajo separados para diferentes cuentas o servicios
- Quieres que Nora funcione como un navegador diario persistente en lugar de una herramienta móvil rápida

## Qué esperar

El escritorio mantiene el mismo modelo de producto global que el móvil:

- Abre los sitios sociales compatibles dentro de una aplicación
- Utiliza la separación de cuentas y las herramientas de navegación en una pantalla más grande
- Instala paquetes nativos para Linux, macOS o Windows
