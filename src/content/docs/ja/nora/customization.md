Nora はカスタム CSS をサポートしており、埋め込まれたサービスウェブサイトの見た目を変更できます。

## 一般的な用途

- 気を散らす UI を非表示にする
- タイムライン中心のレイアウトの密度を向上させる
- 使用しないサービス固有のパーツを削除する
- デスクトップのマルチカラム設定をスキャンしやすくする

## 変更の保守性を維持する

Nora は外部サービスのライブウェブサイトに依存しているため、アップストリームのマークアップは予告なく変更される可能性があります。

保守的なアプローチを推奨します：

- 小さく、ターゲットを絞ったルールを優先する
- サービスの再設計後に再テストする
- 影響を制御できない限り、広範なオーバーライドを避ける

## CSS とプロフィールの組み合わせ

異なるサービスや役割ごとに個別のプロフィールを使用している場合は、動作の予測可能性を維持するために、CSS の調整をそれらのプロフィールに合わせて行ってください。
