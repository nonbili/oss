Nora obsługuje niestandardowy CSS, dzięki czemu możesz zmieniać wygląd osadzanych stron internetowych serwisów.

## Typowe Zastosowania

- Ukrywanie rozpraszających elementów UI
- Poprawa czytelności w układach z gęstą osią czasu
- Usuwanie zbędnych elementów serwisu, których nigdy nie używasz
- Ułatwienie skanowania wzrokiem wielokolumnowych układów desktopowych

## Dbaj o Łatwość Utrzymania Zmian

Ponieważ Nora zależy od dynamicznych stron zewnętrznych serwisów, ich kod źródłowy może ulec zmianie bez ostrzeżenia.

Stosuj konserwatywne podejście:

- Preferuj małe, precyzyjne reguły
- Testuj ponownie po zmianie wyglądu serwisu
- Unikaj szerokich modyfikacji, chyba że kontrolujesz ich skutki

## Łącz CSS z Profilami

Jeśli używasz oddzielnych profili dla różnych serwisów lub ról, dopasuj modyfikacje CSS do tych profili, aby zachowanie aplikacji pozostało przewidywalne.
