Nora är en webbläsare för Android, iOS och desktop optimerad för sociala nätverkstjänster.

Den behåller de ursprungliga webbplatserna för tjänsterna samtidigt som den lägger till native bekvämligheter som separat kontolagring, blockeringslistor, nedladdningsverktyg, URL-rensning och CSS-anpassning.

## Höjdpunkter

- Inga annonser
- Flera konton med separat cookie-lagring
- Stöd för blockeringslistorna EasyList och EasyPrivacy
- Nedladdning av media
- Markera text och zooma
- Ta bort spårningsparametrar från URL:er
- Stöd för anpassad CSS

## Första Körningen

En vettig första inställning är:

1. Öppna den tjänst du använder oftast.
2. Logga in och verifiera att webbflödet fungerar normalt inuti Nora.
3. Skapa ytterligare profiler om du vill ha strikt separation mellan konton.
4. Aktivera blockeringslistor och testa media-nedladdningsbeteende på de tjänster du bryr dig om.
5. Lägg till anpassad CSS först efter att det grundläggande webbläsarflödet är stabilt.

## Hur Det Fungerar

Nora wrappar tjänsternas webbplatser istället för att bygga om varje nätverk som en native klient.

Det tillvägagångssättet håller webbplatserna bekanta samtidigt som Nora kan lägga till:

- Annonsblockering
- Cookie-isolering mellan konton
- Media-nedladdningsflöden
- Tjänstespecifika förbättringar
