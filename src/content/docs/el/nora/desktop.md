Το Nora κυκλοφορεί ως εφαρμογή desktop για Linux, macOS και Windows.

## Τι Προσθέτει το Desktop

Η έκδοση desktop προσθέτει μια διάταξη σχεδιασμένη για μεγαλύτερες οθόνες, συμπεριλαμβανομένης μιας προβολής deck για την περιήγηση σε πολλαπλά timelines πιο άνετα.

Αυτό καθιστά το desktop την καλύτερη επιλογή όταν:

- Παρακολουθείτε πολλά feeds ταυτόχρονα
- Διατηρείτε ξεχωριστούς χώρους εργασίας ανοιχτούς για διαφορετικούς λογαριασμούς ή υπηρεσίες
- Θέλετε το Nora να λειτουργεί ως μόνιμο καθημερινό πρόγραμμα περιήγησης αντί για ένα γρήγορο εργαλείο κινητού

## Τι να Περιμένετε

Το desktop διατηρεί το ίδιο συνολικό μοντέλο προϊόντος με το κινητό:

- Άνοιγμα υποστηριζόμενων κοινωνικών ιστότοπων μέσα σε μία εφαρμογή
- Χρήση διαχωρισμού λογαριασμών και εργαλείων περιήγησης σε μεγαλύτερη οθόνη
- Εγκατάσταση native πακέτων για Linux, macOS ή Windows
