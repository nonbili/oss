Nora admite CSS personalizado para que puedas remodelar los sitios web de los servicios que integra.

## Usos comunes

- Ocultar la interfaz de usuario que distrae
- Mejorar la densidad para diseños con muchas líneas de tiempo
- Eliminar elementos específicos del servicio que nunca usas
- Hacer que las configuraciones de varias columnas en escritorio sean más fáciles de escanear

## Mantén los cambios manejables

Dado que Nora depende de los sitios web en vivo de servicios externos, el marcado original puede cambiar sin previo aviso.

Utiliza un enfoque conservador:

- Prefiere reglas pequeñas y específicas
- Vuelve a realizar pruebas después de los rediseños del servicio
- Evita las anulaciones generales a menos que controles las consecuencias

## Combina CSS con perfiles

Si utilizas perfiles separados para diferentes servicios o roles, mantén los ajustes de CSS alineados con esos perfiles para que el comportamiento sea predecible.
