У NouTube есть сборка для рабочего стола для Linux, macOS и Windows.

## Чего ожидать

Десктопная версия сохраняет ту же общую идею, что и мобильное приложение:

- YouTube и YouTube Music в одном приложении
- Нативные установщики для Linux, macOS и Windows
- Окно на рабочем столе для длительных сеансов прослушивания или просмотра

## Зачем использовать десктопную версию

Выбирайте десктопную версию, если вам нужны:

- Нативные пакеты для рабочего стола
- Воспроизведение в окне
- Платформенные процессы установки для Linux, macOS или Windows

## Когда лучше выбрать десктоп

Десктопная версия больше подходит, если вы хотите использовать NouTube как ежедневное приложение для рабочей станции, а не как плеер в первую очередь для телефона.
