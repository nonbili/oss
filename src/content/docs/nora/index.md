Nora is an Android, iOS, and desktop browser optimized for social network services.

It keeps the original service websites while adding native conveniences like separate account storage, blocklists, download tools, URL cleanup, and CSS customization.

## Highlights

- No ads
- Multiple accounts with separate cookie storage
- EasyList and EasyPrivacy blocklist support
- Media downloads
- Select text and zoom
- Remove tracking query parameters from URLs
- Custom CSS support

## Platforms

- Android
- iOS
- Desktop

## First Run

A sensible first setup is:

1. Open the service you use most often.
2. Sign in and verify that the web flow works normally inside Nora.
3. Create additional profiles if you want strict separation between accounts.
4. Enable blocklists and test media download behavior on the services you care about.
5. Add custom CSS only after the baseline browsing flow is stable.

## How It Works

Nora wraps the service websites instead of rebuilding each network as a native client.

That approach keeps the sites familiar while letting Nora add:

- Ad blocking
- Cookie isolation between accounts
- Media download flows
- Service-specific enhancements
