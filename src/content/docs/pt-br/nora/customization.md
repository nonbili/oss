O Nora suporta CSS personalizado para que você possa remodelar os sites de serviços que ele incorpora.

## Usos Comuns

- Ocultar interface que distrai
- Melhorar a densidade para layouts com muitas timelines
- Remover componentes específicos do serviço que você nunca usa
- Facilitar a varredura visual em configurações multi-colunas no desktop

## Mantenha as Alterações Sustentáveis

Como o Nora depende dos sites ao vivo de serviços externos, a marcação original pode mudar sem aviso prévio.

Use uma abordagem conservadora:

- Prefira regras pequenas e direcionadas
- Teste novamente após redesenhos do serviço
- Evite substituições amplas, a menos que você controle as consequências

## Combine CSS com Perfis

Se você usa perfis separados para diferentes serviços ou funções, mantenha os ajustes de CSS alinhados com esses perfis para que o comportamento permaneça previsível.
