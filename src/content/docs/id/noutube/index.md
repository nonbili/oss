NouTube adalah aplikasi Android dan desktop untuk YouTube dan YouTube Music.

Aplikasi ini membungkus aplikasi web seluler YouTube dan YouTube Music dan melapisi perilaku native di atasnya, termasuk pemblokiran iklan, hook kontrol pemutaran, dukungan unduhan, RSS feed, dan manajemen pustaka yang ramah luring.

## Sorotan

- Tanpa iklan
- Pemutaran latar belakang
- Dukungan video dan musik dalam satu aplikasi
- Manajemen pustaka dan riwayat tanpa masuk (sign in)
- Unduhan video
- Pembaca RSS feed untuk saluran YouTube
- Dukungan live chat
- Dukungan CSS kustom

## Platform

- Android
- Desktop

Build desktop mengikuti model produk yang sama dengan aplikasi seluler, dengan installer native untuk layar yang lebih besar.

## Penggunaan Pertama

Setelah instalasi, penyiapan pertama yang praktis terlihat seperti ini:

1. Buka YouTube atau YouTube Music di dalam aplikasi.
2. Periksa perilaku pemutaran, terutama pemutaran latar belakang jika itu adalah kegunaan utama Anda.
3. Konfigurasikan pustaka, riwayat, feed, dan preferensi unduhan Anda.
4. Tambahkan CSS kustom hanya setelah memastikan pengalaman default berfungsi untuk Anda.

## Tautan yang Didukung

NouTube dikonfigurasi untuk menangani host ini:

- `youtube.com`
- `www.youtube.com`
- `m.youtube.com`
- `music.youtube.com`
- `youtu.be`

Jika perangkat Anda menanyakan aplikasi mana yang harus membuka tautan yang didukung, pilih `NouTube`.

## Cara Kerjanya

NouTube membungkus aplikasi web alih-alih membangun kembali YouTube dari awal. Dalam praktiknya itu berarti:

- Anda mempertahankan alur dan tata letak situs resmi
- Aplikasi menambahkan pemblokiran iklan dan hook pemutaran native
- Pemutaran latar belakang dan integrasi perangkat berfungsi lebih baik daripada menggunakan situs di browser biasa
