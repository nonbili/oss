يدعم NouTube لغة CSS المخصصة لصفحات YouTube و YouTube Music المدمجة.

## فائدة CSS المخصص

- إخفاء العناصر التي لا تهتم بها
- زيادة كثافة الصفحة
- تعديل التباعد أو الخطوط
- الالتفاف حول خيارات واجهة المستخدم للموقع التي تعيق طريقك

## النهج الموصى به

اجعل التغييرات ضيقة وسهلة الإزالة.

- ابدأ بقاعدة أو قاعدتين مستهدفتين
- فضل المحددات المستقرة حيثما أمكن ذلك
- أعد الاختبار بعد تغييرات تخطيط الموقع الأصلي

نظرًا لأن NouTube يغلف خصائص ويب مباشرة، يمكن أن يتلف CSS المخصص عندما يغير YouTube ترميزه.

## قبل التخصيص

تأكد من أن سلوك التطبيق الافتراضي يعمل بالفعل من أجل:

- التشغيل
- التنقل
- الخلاصات
- التنزيلات

بمجرد استقرار الأساس، أضف تغييرات CSS تدريجيًا.
