Nora 作为桌面应用发布，支持 Linux, macOS, 和 Windows。

## 桌面端新增功能

桌面版本增加了为大屏幕设计的布局，包括可以更舒适地浏览多个时间线的面板视图 (Deck View)。

这使得桌面端更适合以下场景：

- 同时监控多个 Feed
- 为不同的账号或服务保持独立的对应工作区
- 希望 Nora 作为一个持久的日常浏览器，而不是一个快速的移动端工具

## 预期效果

桌面端保持与移动端相同的整体产品模型：

- 在一个应用内打开支持的社交网站
- 在更大的屏幕上使用账号分离和浏览工具
- 为 Linux, macOS, 或 Windows 安装原生安装包
